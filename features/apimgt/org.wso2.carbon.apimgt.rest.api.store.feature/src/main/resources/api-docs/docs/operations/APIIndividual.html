Get Details of API
GET https://apis.wso2.com/api/am/store/v0.9/apis/{apiId}

Using this operation, you can retrieve complete details of a single API. You need to provide the Id of the API to retrive it. `X-WSO2-Tenant` header can be used to retrive an API of a different tenant domain. If not specified, the user's tenant associated with the access token will be used.

Not required

Get the swagger of an API
